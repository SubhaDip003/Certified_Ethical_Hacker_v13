# 🌐 Network-Level Session Hijacking
**Network-Level Session Hijacking** is a method where attackers take control of a user's session by targeting the communication between devices on a network. Unlike other types, it doesn’t need access to the victim's device or specific application code. Instead, attackers hijack the session by exploiting transport and internet protocols. This allows them to capture important session data, which can later be used to launch deeper attacks at the application level. It’s a powerful and stealthy technique often used to gain unauthorized access without detection.

The following are different types of network-level hijacking: 
```
- Blind hijacking	- RST hijacking
- UDP hijacking		- Man-in-the-middle: packet sniffer
- TCP/IP hijacking	- IP spoofing: source routed packets
```
